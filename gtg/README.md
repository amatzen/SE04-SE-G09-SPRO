# Controls
| Controls           | Description                   |
|--------------------|-------------------------------|
| "A" or Left arrow  | Turn left                     |
| "D" or Right arrow | Turn right                    |
| "W" or Up arrow    | Accelerate                    |
| "S" or Down arrow  | Decelerate                    |
| Space              | Shoot                         |
| "M"                | Mute/unmute music             |
| "K"                | Suicide (Debug)               |
| "P"                | Increase Wanted level (Debug) |
| "G"                | God mode (Debug)              |
| "H"                | Refill health (Debug)         |
